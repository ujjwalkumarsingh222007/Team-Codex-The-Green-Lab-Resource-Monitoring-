# 🌱 Green Lab – SDG Smart Resource Monitoring Platform

## Udbhava X ’26 Hackathon Submission Repository

This repository contains our official hackathon submission for **Udbhava X ’26**, organized by the **Coding Club, School of Computer Science & Artificial Intelligence, SR University**.

## 🚀 Project Overview

**Green Lab** is an AI-powered smart campus sustainability platform that monitors **water, gas, and electricity usage in real time**, detects leaks and anomalies, enables **role-based dashboards (admin, faculty, student)**, supports **resource requests and emergency alerts**, and tracks **carbon emissions with SDG impact analytics**.

### 🌍 SDG Goals Supported

* SDG 6 – Clean Water and Sanitation
* SDG 7 – Affordable and Clean Energy
* SDG 12 – Responsible Consumption and Production
* SDG 13 – Climate Action
